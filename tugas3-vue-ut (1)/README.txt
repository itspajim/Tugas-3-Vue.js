Tugas 3 - SITTA UT (Vue.js)
Project sederhana yang memenuhi persyaratan Tugas Praktik 3.

Cara menjalankan:
1. Extract zip ke folder server lokal.
2. Jalankan dengan HTTP server karena fetch() membutuhkan origin (misal: python -m http.server).
3. Buka index.html di browser.

Isi:
- index.html
- assets/css/style.css
- data/dataBahanAjar.json (sudah termasuk)
- js/services/api.js
- js/components/* (stock-table, do-tracking, status-badge, app-modal)
- js/app.js
